# Dorf Event App

Ein erster lauffaehiger Prototyp fuer eine lokale Dorf-Event-App.

## Start in PyCharm

1. Projektordner in PyCharm oeffnen.
2. Python-Interpreter auswaehlen oder eine virtuelle Umgebung erstellen.
3. `app.py` starten.
4. Im Browser `http://127.0.0.1:8000` oeffnen.

## Funktionen im Prototyp

- Eventliste mit Datum, Ort, Kategorie und Beschreibung
- Kartenansicht mit Markern
- Filter fuer Kategorie, Zeitraum und Suchtext
- Interesse-Button pro Event
- API-Endpunkte unter `/api/events`

Der Prototyp nutzt nur Python-Standardbibliothek. Fuer die Karte werden Leaflet und OpenStreetMap-Kacheln im Browser geladen.
